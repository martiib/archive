Client archive

Client Archive Management System (Python + SQLite)
Project Overview

This project is a client archive management system built in Python.  
It allows users to manage client data through a command-line interface using SQLite for storage and Pandas for data display.  
It is an improved version of a previous file-based system, created to practice object-oriented programming and database handling in Python.

Key Features

- Add new clients to the database
- View all clients in table format
- Update and delete client records
- Persistent storage using SQLite
- Simple command-line interface
- Data display using Pandas

Learning Focus

- Object-oriented programming in Python
- SQLite database management
- CRUD operations (Create, Read, Update, Delete)
- Data handling with Pandas
- Writing clean and structured Python code

Technologies Used

- Python 3.x
- SQLite3
- Pandas

Project Structure

- Client class – defines client data structure
- Archive class – handles database operations
- run function – manages user interaction

Improvements Over Previous Version

- Replaced text file storage with SQLite database
- Added full CRUD functionality
- Improved code structure using object-oriented programming
- Better scalability and data handling
